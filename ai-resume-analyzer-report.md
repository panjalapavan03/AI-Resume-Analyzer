# AI Smart Resume Analyzer

## Objective
Analyze resumes using NLP and Machine Learning by comparing them with a job description.

## Features
- PDF Resume Upload
- Text Extraction
- Skill Detection
- Resume Match Score
- Dashboard
- Suggestions

## Algorithm
TF-IDF Vectorization + Cosine Similarity

## Technologies
Python, Streamlit, Scikit-learn, PyPDF2, Matplotlib
